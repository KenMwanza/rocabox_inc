# Rocabox Inc - FedEx Ground Contractor Website

A professional one-page website for Rocabox Inc, a FedEx Ground contractor based in Watertown, NY.

## 🚀 Live Website
The website is a single HTML file that can be hosted anywhere. It includes:
- Modern, responsive design
- Complete information about Rocabox Inc
- Contact form functionality
- Mobile-friendly layout

## 📋 Features
- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design with gradient accents
- **Complete Information**: All details about Rocabox Inc from the database
- **Contact Form**: Functional contact form (frontend only)
- **Fast Loading**: All CSS and JavaScript inline for fast loading
- **No Dependencies**: Only uses CDN for Font Awesome and Google Fonts

## 🎨 Design Elements
- **Color Scheme**: Purple/indigo primary with green accents
- **Typography**: Poppins for headings, Roboto for body text
- **Icons**: Font Awesome icons throughout
- **Sections**:
  1. Hero section with call-to-action
  2. About section with company details
  3. Services section highlighting AVP and P&D
  4. Contact section with form and contact info

## 🛠️ Technical Details
- **File**: `index.html` (single file)
- **CSS**: All styles inline in `<style>` tag
- **JavaScript**: Form handling in `<script>` tag
- **External Resources**:
  - Font Awesome 6.4.0 (icons)
  - Google Fonts (Poppins, Roboto)
- **No build process required**

## 📱 Responsive Breakpoints
- **Desktop**: 1200px max width
- **Tablet**: 768px and below
- **Mobile**: 480px and below

## 🚀 Deployment
This website can be deployed to:
1. **GitHub Pages** (enabled in repository settings)
2. **Netlify** (drag and drop the HTML file)
3. **Vercel** (import the repository)
4. **Any static hosting** (AWS S3, Firebase, etc.)

## 🔧 Customization
To customize this website:
1. Edit the `index.html` file
2. Update contact information in the contact section
3. Modify colors in the CSS `:root` variables
4. Add/remove sections as needed
